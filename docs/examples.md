# 範例程式 — 照著拖就會動

每個範例都從「🟢 當程式開始」帽形積木開始（從左側「🟢 開始與重置」分類拖出），底下接動作。

---

## 範例 1：Hello Bittle（最簡單）

打招呼 + 站穩。

```
🟢 當程式開始
  └─ 👋 打招呼
  └─ 🧍 站穩
```

**操作**：
1. 從「🟢 開始與重置」拖「🟢 當程式開始」到中央
2. 從「🐕 動作」拖「👋 打招呼」接到下方
3. 再拖「🧍 站穩」接到下方
4. 按右上角 **▶ 執行**

**預期**：模擬器顯示 RF 腿揮 3 次，然後所有腿歸零

---

## 範例 2：完整走路週期

站穩 → 走 → 等 → 坐 → 等 → 站起來。

```
🟢 當程式開始
  └─ 🧍 站穩
  └─ ⏱️ 等待 1 秒
  └─ 🚶 走向前
  └─ ⏱️ 等待 2 秒
  └─ 🪑 坐下
  └─ ⏱️ 等待 2 秒
  └─ 🧍 站穩
```

**預期**：依序看到站立 → 走路擺動 → 坐姿（後腿大彎）→ 重新站立

---

## 範例 3：重複動作

坐下 / 站穩交替 3 次（用「重複 N 次」迴圈）。

```
🟢 當程式開始
  └─ 🔁 重複 [3] 次
       └─ 🪑 坐下
       └─ ⏱️ 等待 1 秒
       └─ 🧍 站穩
       └─ ⏱️ 等待 1 秒
```

**操作**：
- 「重複 N 次」在左側 **🔁 流程控制** 分類
- 把 4 個積木（坐下 → 等 → 站穩 → 等）疊在迴圈內側

---

## 範例 4：個別 Servo 控制（頭部搖擺）

連續控制 servo 0（head pan），讓頭左右搖動。

```
🟢 當程式開始
  └─ ⚙️ 移動 servo [0 (頭)] 到 [30] 度
  └─ ⏱️ 等待 0.5 秒
  └─ ⚙️ 移動 servo [0 (頭)] 到 [-30] 度
  └─ ⏱️ 等待 0.5 秒
  └─ ⚙️ 移動 servo [0 (頭)] 到 [0] 度
  └─ 🔄 重置 Bittle 到站穩姿勢
```

**預期**：頭順時針旋轉 30 度 → 反向 30 度 → 回正 → 全身 reset

對應的實機 ASCII 指令（執行時會在右下「執行記錄」看到）：
```
m 0 30
m 0 -30
m 0 0
kbalance
```

## 範例 6：條件邏輯（搭配變數）

從「📦 變數」建立一個變數 `count`，迴圈計數。

```
🟢 當程式開始
  └─ 設定 [count] 為 [0]
  └─ 🔁 重複 [5] 次
       └─ 🚶 走向前
       └─ 設定 [count] 為 [count + 1]
       └─ 顯示 [count]
```

可以從「🧮 數學」拿「+」算術積木，從「📦 變數」設定 / 讀取。

## 故障排除

| 症狀 | 原因 | 解法 |
|---|---|---|
| 按 ▶ 執行沒反應 | Workspace 是空的 | 拖至少 1 個 hat block + 1 個動作積木 |
| 「Workspace 是空的」警告 | 積木沒接到 hat block | 從「🟢 開始與重置」拖「當程式開始」當頂端 |
| 「Blockly.JavaScript generator 未載入」 | index.html 缺 javascript_compressed.js | 已修，按 Ctrl+F5 強制重整 |
| 看到「執行錯誤」 | Code 跑出異常 | F12 → Console 看詳細 stack trace |
| SVG 看不到狗形 | CSS / SVG 結構衝突 | 已修，Ctrl+F5 重整 |
| Web Serial 連不上 | 必須 http://localhost（不能 file://） | 用 VS Code Live Server 或 Python http server |

## 看得懂執行記錄就會除錯

當您按 ▶ 執行，右下「執行記錄」應該會看到：

```
[時間] --- 生成的 JS code ---
[時間]   await BittleApp.runtime.send('khi');
[時間]   await BittleApp.runtime.send('kbalance');
[時間] === 開始執行 ===
[時間] ▶ 送出: khi
[時間] ▶ 送出: kbalance
[時間] === 執行完成 ===
```

如果看到 `--- 生成的 JS code ---` 但下面是空的，就是積木沒生成 code（多半是沒接到 hat block）。

如果連 `--- 生成的 JS code ---` 都沒看到，就是 Run 按鈕沒接上或 generator 未載入。
